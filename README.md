
# ClassicAutoRepair v1.0

## Installation

### Github

Move folder ClassicAutoRepair (it should be named as it) to your Wow Classic Addons folder location. If you downloaded the zip file from Github you should rename it by removing the source tag attached to it like "-master" or "-1.0" for example.

### WowCurse

Download from [https://www.curseforge.com/wow/addons/classicautorepair](https://www.curseforge.com/wow/addons/classicautorepair) and simply move ClassicAutoRepair folder to your Wow  Classic Addons folder location.
